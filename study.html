The language of the web!

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of a webpage and allows you to include text, images, links, and more.

HTML Attributes

  • href: Specifies the URL for a link.
  • src: Specifies the source for an image or a script.
  • alt: Provides alternative text for images.
  • id: Specifies a unique ID for an element.
  • class: Assigns a class name to an element for styling.
  • style: Adds inline CSS styles to an element.
  • title: Provides additional information as a tooltip.

Learn more about HTML and its attributes at W3Schools.
